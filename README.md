# C# .NET 7 and Angular app

## Prerequisites

- dotnet sdk (v7)
- node.js (v18)
- [Angular 16 CLI](https://angular.io/guide/setup-local)
  - `npm install -g @angular/cli@16`

## Starting the app

### Starting the client:

- `cd client`
- `ng serve`

### Starting the server:

- open a terminal
- `cd API`
- `dotnet watch` or `dotnet run`
